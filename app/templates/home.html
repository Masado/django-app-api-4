Analysis Pipelines Web Application

The Analysis Pipelines Web Application is a tool designed to make running analyses an easy endeavor.
It allows you to choose the kind of data you produced and what kind of analysis you would like to run on it. Uploading your data and providing general information will allow the webapp to construct the nextflow command to call the analysis pipeline.
The analysis will be run for you and the results will be provided once completed.
